# utils
Importable molecular dynamics analysis utilities, including Native contacts, contact probability maps, Free energy, heat capacity and energy landscapes calculated via the WHAM algorithm. (Kumar, et Al. 92) 
Designed primarily for used with the Go-Model Pressure Project the code is easily modified for standard MD output file types to fit with Gromacs and OpenMM.

Import Usage: 
    sudo wget https://raw.githubusercontent.com/twatchorn/utils/main/utils.py
    import utils
